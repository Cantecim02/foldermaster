# Editio legal pages

These static pages are ready to replace the current public legal routes:

- `privacy/index.html` -> `https://www.editioapp.com/privacy/`
- `terms/index.html` -> `https://www.editioapp.com/terms/`
- `support/index.html` -> `https://www.editioapp.com/support/`

Deploy the directories together with `legal.css`. Verify all three HTTPS URLs before enabling account registration or submitting the account-enabled build to App Store review.
